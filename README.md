# Spring Boot宠物管理系统 java539

## 项目概述

宠物管理系统是一个基于Spring Boot和Vue的前后端分离项目，主要用于处理宠物领养、流浪宠物救助等事务。

## 技术栈

- 后端：Spring Boot、MyBatis-plus、MySQL
- 前端：Vue、elementUI
- 开发工具：IDEA、Node.js、Maven

## 系统角色

- 管理员：负责后台管理操作。
- 用户：进行宠物领养、信息交流等。

## 运行环境

- JDK 8
- MySQL 8.0
- IDEA
- Node.js 14
- Maven

## 功能模块

### 用户端功能

- **首页**：
  - 展示公告列表、宠物科普、流浪宠物介绍和热门活动。
- **宠物领养**：
  - 用户可搜索、申请领养宠物，并查看自己领养的宠物。
- **宠物救助**：
  - 展示需要救助的流浪宠物信息，用户可新增流浪宠物信息，查看救助站信息。
- **丢失宠物查看**：
  - 展示其他用户丢失的宠物和失主信息。
- **宠物论坛**：
  - 用户交流宠物领养心得，评论他人帖子。

### 管理员端

- **主页**：
  - 展示注册人数统计，各类图表数据。
- **用户管理**：
  - 管理注册用户信息。
- **角色管理**：
  - 分配用户系统菜单。
- **菜单管理**：
  - 管理后台菜单。
- **文件管理**：
  - 管理系统文件。
- **流浪动物管理**：
  - 发布流浪宠物信息。
- **待绝育动物管理**：
  - 管理流浪动物健康信息。
- **申请领养管理**：
  - 处理用户领养信息。
- **评论管理**：
  - 管理论坛用户评论。
- **流浪动物救助**：
  - 发布救助信息，处理已解决救助。
- **喂养点管理**：
  - 管理宠物喂养点。
- **动物走失管理**：
  - 管理丢失动物信息。
- **救助站管理**：
  - 管理流浪动物救助站。
- **帖子管理**：
  - 管理宠物论坛帖子。
- **捐赠管理**：
  - 管理捐赠人信息。
- **公告管理**：
  - 管理用户端首页公告。
- **科普文章管理**：
  - 管理用户端科普文章展示。
- **活动管理**：
  - 管理宠物相关活动。

## 目录结构

```plaintext
|-- pet_system
```

## 核心亮点

- 使用Vue和Spring Boot的前后端分离架构，易于维护和扩展。
- 基于MyBatis-plus，简化数据库操作。
- 使用elementUI，提供友好的用户界面。
- 功能模块划分清晰，覆盖宠物领养到管理的全流程。
